HTML

HyperText Markup Language

HTML is the standard program for coding. It stands for HyperText Markup Language, it’s used for creating, designing, and organizing websites or web pages. It also contains many elements that help build websites.

CSS

Cascading Style Sheets

CSS stands for Cascading Style Sheets. It is a program or extention used in coding, particularly provides more options in designing or styling a website. You may also work on CSS properties either internally – found in the head section – and externally – can be accessed via creating a link and another file.

Bootstrap Version 4

Bootstrap is a program connected to HTML, as well as CSS properties. It is greatly known for its Easy-To-Use features. It is widely open to different devices for itt provides 4 containers; “.container-sm”, “.container-md”, “.container-lg”, and “.container-xl”.
